Amazon Web Serviceとは、Amazon社が提供するクラウド機能の多数のサービス群の総称です。多数の便利なサービスがあり、柔軟な利用が可能なことから、企業での本格的な主要アプリケーションから個人レベルの小さなサイトまで幅広く使われています。

Amazonと言えば、GAFAの一角でもある最も巨大なIT企業の一つで、eコマースでの通信販売でも有名ですが、このAWSといったクラウド事業の規模も大きいです。

高い冗長性も特徴で、99.9…%といったサービスレベル保証(SLA)をしているなど、高い可用性が確保されています。それでも大規模障害などは起きることがありますが、複数拠点での冗長構成をしてより堅牢なシステムを構築することもできます。
