- 意味

\[名詞\] プログラミングとは、コンピュータ上で動かすプログラムを作成する事。プログラミング言語によって人間の手によって書かれるもの。

- 説明

現代のアプリケーションはほとんどがプログラミング言語を用いてプログラミングされたものである。プログラミング言語には様々な種類があり、昔からあるものや、新たに開発されるものも多い。有名所を以下に列挙する。

- HTML
- Python
- Java
- C
- Ruby

一般的なイメージとして、専門性が高く非常に難しいものと認識されていることが多い。プログラミングと言われると以下の画像のようなものを想像して毛嫌いされたりもする。

![programming](images/programming-300x198.jpg)

実際にはこのような意味不明な文字列ではなく、英語が読めればどのようなプログラムなのかが分かるようなものも多い。なぜなら、そもそもプログラミング言語とは、「人間が分かりやすく書けるように」作られた言語なのであるからである。究極的には以下もプログラミング言語と言え、非常に分かりやすいものも存在する。

![programming2](images/programming2-300x122.jpg)
